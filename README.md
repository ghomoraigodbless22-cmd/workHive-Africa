# WorkHive Africa

WorkHive Africa is a digital marketplace connecting freelancers, students, artisans, and local workers with individuals and businesses seeking services across Africa.

## Features

- User Registration
- Worker Profiles
- Job Posting
- Job Applications
- Messaging
- Reviews and Ratings

## Target Users

- Students
- Freelancers
- Artisans
- Skilled Professionals
- Local Laborers
- Small Businesses

## Vision

To become Africa's leading platform for connecting talent with opportunity, empowering workers and helping clients find trusted service providers.

## Technology Stack

- Flutter
- Firebase
- GitHub
